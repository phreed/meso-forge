= asciidoctor-pdf
:version: 2.3.19

An AsciiDoc to PDF processor

== Description

A fast, open source text processor and publishing toolchain for converting AsciiDoc content to PDF.

```bash
pixi global install -c meso-forge asciidoctor-pdf
```
